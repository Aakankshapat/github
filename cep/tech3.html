    AICTE Post Graduate / GATE-based Scholarship

      Eligibility Criteria:

      • Must have qualified GATE / GPAT as required.
      • Enrolled in full-time postgraduate technical programme.
      • Other criteria (income, academic performance) as per AICTE scheme.

      Requried Document:

      • GATE / GPAT score / certificate.
      • Admission / enrollment proof in PG program.
      • Academic transcripts / previous degree certificates.
      • Income Certificate.
      • Identity proof (Aadhaar, etc.).
      • Bank account details (with IFSC, etc.).

      Application guidance

      Eligible applicants can apply by following these steps:-
      step-1: Go to AICTE Students Development Schemes page.
      step-2: Locate the scholarship option for PG / GATE students.
      step-3: Register / login.
      step-4: Fill the application form.
      step-5: Upload required documents.
      step-6: Submit and note application acknowledgment.
      step-7: Check status on portal.
